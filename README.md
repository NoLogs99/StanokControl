# 🚀 StanokControl

**Современная система управления и мониторинга промышленных станков**

> Telegram-бот + FastAPI-бэкенд + веб-панель для контроля производственного оборудования в реальном времени.  
> Поддержка Postgres, Redis-кэширования, SQLModel ORM и Sentry-мониторинга.

---

## ✨ Возможности

- ⚡ Управление станками через удобный Telegram-бот  
- 🌐 REST API на FastAPI с автоматической документацией Swagger  
- 🗄️ Надёжное хранение данных в PostgreSQL  
- ⚡ Молниеносный кэш и очереди на Redis  
- 🔍 Полный мониторинг ошибок через Sentry  
- 📊 Красивая веб-панель (HTML + CSS + JavaScript)  
- 📦 Pydantic-валидация и типизация на всех уровнях
